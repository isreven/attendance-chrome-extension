# Debug Steps

1. Go to chrome://extensions/
2. Find "SAP Synerion Attendance Updater"
3. Click "service worker" link (should be under the extension name)
4. A DevTools window will open showing background script console
5. Look for errors in red
6. Share any error messages you see

Also check:
- Is the service worker status "active" or "inactive"?
- Do you see "🔧 Background service worker started" in the console?
