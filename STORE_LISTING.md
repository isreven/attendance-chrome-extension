## Store Listing Information

### Extension Name
**SAP Synerion Attendance Updater**

### Short Description (132 characters max)
Automate updating attendance codes in SAP Synerion. One-click remote work updates or custom date/code assignments.

### Detailed Description

Streamline your SAP Synerion attendance management with automated activity code updates.

**Key Features:**

🏠 **One-Click Remote Work**
Update all your attendance dates to "Remote Work" (code 27) with a single click. Perfect for regular remote workers.

🎯 **Custom Date Builder**
Need different codes for specific dates? Use our visual date builder:
- Add multiple dates with individual activity codes
- Date picker and dropdown interface
- All other dates default to Remote Work
- Perfect for vacation days, client visits, or sick leave

💾 **Automatic Saving**
The extension automatically clicks the save button after each update—no manual intervention needed.

⚡ **Lightning Fast**
Uses your existing browser session—no separate login required. Updates happen instantly in your active tab.

📋 **Full Code Support**
Supports all 29 SAP Synerion activity codes:
- Remote Work (27)
- Client Site (36)
- Vacation (2)
- Sick Leave (11)
- And 25 more...

**How It Works:**

1. Navigate to SAP Synerion Daily Browser (while logged in)
2. Click the extension icon
3. Choose "Update All" or build custom date assignments
4. Watch as your attendance is updated automatically
5. All changes are saved automatically

**Smart Features:**
- Only updates the active dropdown per row
- Skips rows that already have the correct code
- Handles multiple InOut entries correctly
- Real-time progress updates
- Detailed console logging for troubleshooting

**Privacy & Security:**
- No data collection or tracking
- Uses your existing browser session
- All processing happens locally
- No credentials stored
- Open source code

Perfect for SAP employees who want to save time on attendance management!
